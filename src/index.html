L'Œuvre & Le Commun

SoundSpot est une infrastructure sonore décentralisée pour l'espace public. Il transforme un Raspberry Pi Zero 2W en un routeur audio WiFi (Snapcast + Icecast) totalement autonome.

C'est un "Bien Commun" : n'importe quel passant peut s'y connecter en WiFi pour écouter la session musicale en mode "Silent Disco" via son casque, et n'importe quel DJ peut s'y connecter pour prendre l'antenne et diffuser son mix en temps réel sur les enceintes solaires du lieu.

Ne détruisez pas les SoundSpots que vous croisez dans la rue. Dupliquez-les ! Ce code est libre et vous donne le pouvoir de créer votre propre radio locale.

Architecture Technique V2

  • Réseau : WiFi AP (hostapd), Portail Captif (OpenNDS v9+)
  • Audio : Mixxx Live Broadcasting → Icecast2 → Ffmpeg → Snapcast → PipeWire
  • IA Edge : OpenCV Haar Cascade (Détection de présence pour message d'accueil)
  • Hardware : RPi Zero 2W, Pi Camera V3, Capteur batterie INA219, Enceinte Bluetooth

Récupérer & Déployer

Pour déployer votre propre nœud, vous avez besoin d'un PC Linux (Debian/Ubuntu) connecté à un réseau WiFi qui servira de base. Depuis un terminal :

# 1. Cloner le code source (via IPFS, IPNS ou Github)
git clone https://github.com/papiche/sound-spot
cd sound-spot

# 2. Brancher une carte SD vierge et lancer l'installeur interactif
bash deploy_soundspot.sh

# L'installeur va flasher l'OS, configurer le réseau, installer Mixxx
# sur votre PC, et préparer la carte SD. Insérez-la ensuite dans le Raspberry Pi !

Une fois la carte insérée dans le Raspberry Pi et l'alimentation branchée, le nœud s'auto-configure en 10 minutes. Lancez ensuite la commande ~/zicmama_play.sh sur votre PC pour commencer à diffuser !
